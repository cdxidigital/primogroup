## 🎨 Design Specifications

### Color Palette
```
Forest Green (#1B4332)  → Primary buttons, process badges
Gold (#D4AF37)          → Section accents, premium feel
Cream (#F5F1E8)         → Primary background
Charcoal (#2C2C2C)      → Text and headings
Sage Green (#52B788)    → Hover states
Warm Beige (#FAF6F0)    → Card backgrounds
```

### Typography
```
Headlines: Serif font (72px down to 24px)
Body Text: Sans-serif (16px standard)
Line Height: 1.4-1.6
Font Pairing: Professional + Modern
```

### Responsive Breakpoints
```
Mobile:    < 768px    (Single column, hamburger menu, bottom nav)
Tablet:    768-1024px (2 columns, adjusted spacing)
Desktop:   1024px+    (Full multi-column, advanced features)
```
